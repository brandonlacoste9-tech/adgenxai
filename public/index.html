1. Add Credentials

Set your Instagram, YouTube, or TikTok API credentials as environment variables in Netlify.

2. Call the API

Make POST requests to the serverless functions with your content data.

3. Content Goes Live

Your content is automatically published to the platform of your choice.

Example Usage

// Post to Instagram
const response = await fetch('/.netlify/functions/post-to-instagram', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({
    imageUrl: 'https://example.com/image.jpg',
    caption: 'Check out this amazing content! üöÄ'
  })
});

const result = await response.json();
console.log('Posted:', result.publishedId);
